[Tips_01]AutoHotkey & Create a simple dairy log

2015-09-08

Lâu lâu ngồi quay lại sự nghiệp viết blog giới thiệu cái Tip bé bé về tạo một simple Dairy Log nhanh gọn giúp bạn quản lý công việc của mình một cách đơn giản nhất có thể.

Daily Log là gì?

Daily Log là một dạng ghi chú giúp bạn biết được những gì mình đã làm và dự định những việc làm sắp tới trong ngày. Việc ngồi tạo một Daily Log hằng ngày sẽ giúp bạn tập trung vào những mục tiêu mà bạn đặt ra, và đương nhiên nhìn vào đó thì sẽ biết mình cần làm cái gì trong ngày hôm đó.

Những công cụ giúp bạn có thể tạo một Dairy Log hoặc một List các công việc cần được làm như:

  1. Evernote : một dạng note ghi chú những thứ mình cần nhớ có rất nhiều tính năng hữu ích như tạo list, checklist, quản lý bằng category, v.v.. sẳn giới thiệu thêm cái tool của Evernote là Skitch giúp ghi chú lên hình ảnh rất cool và nhanh gọn (những hình ảnh trong bài mình dùng Skitch để tạo đó).
  2. Simple Note : giống Evernote nhưng nó đơn giản hơn rất nhiều
  3. Trello : Cái này là một tool giúp cho bạn quản lý project cũng như tạo các task, list, card, board để quản lý các dự án của mình. Nó dự trên nên tảng của hệ thống KanBan - chia công việc thành từng cột rồi giải quyết từng cái

Những tool trên đểu có thể dùng trên Website, iOS, Android, Window Phone nó đều hỗ trợ App để đồng bộ hoá cả.

Tóm lại, bài này không có viết về 3 cái tools trên để tạo 1 Dairy Log vì thực chất ra dùng những cái trên cho 1 nhóm thì sẽ hiệu quả hơn là cho cá nhân. Nhìn chung bạn phải làm vài bước để tạo 1 cái LOG.

Cái mình hướng đến là một công cụ tạo LOG nhanh và càng đơn giản cảng tốt giống như vầy:

[Hình 1]

Việc tạo LOG chỉ cần bạn nhấn tổ hợp phím Ctrl + Alt + L nó sẽ ra một cái popup

[Hình 2]

Cứ thế nhập vào chương trình sẽ tự động ghi vào file DairyLog.txt cho bạn

Phần cài đặt

Những thứ cần cài đặt

  1. AutoHotKey : Một công cụ giúp lập trình những phím tắt cho các bạn “siêu lười” dùng máy tính =]]

  2. Hết rồi…..

Cách cài đặt:

  • Sau khi bạn cài đặt AutoHotKey xong thì trên dextop tạo 2 files như vầy:

    • DairyLog.txt: dùng để save những Log của bạn.
    • DairyHotKey.ahk: dùng để lập trình cho AutoHotKey

[Hình 3]

  • Trong file DairyHotKey.ahk thì bạn tạo với đoạn code này

    ; Hotkey Ctrl + Alt + L        
^!l::
; Lấy giờ của hệ thống
current=%A_Hour%:%A_Min% %A_DD%/%A_MM%/%A_YYYY%
; Tạo một popup 
InputBox, UserInput, Daily Log, What did you Done?, , 380, 170
If Not ErrorLevel
{
; Thêm text input vào trong file vs đường dẫn
FileAppend,[%current%]`n%UserInput%`n=====`n, C:\Users\ThinkPad-Kid\Desktop\DailyLog.txt
}
Return

[Hình 4]

  • Sau khi xong việc thêm code vào trong file DairyHotKey.ahk thì chỉ còn Run Script AutoHotKey. Bạn chuột phải vào trong file rồi chọn Run Script thôi.

[Hình 5]

  • Bước cuối cùng là nhấn Ctrl + Alt + L và thử ghi gì đó sau đó kiểm tra trong file DairyLog.txt nhé.

Việc tạo Dairy Log sẽ giúp bạn biết được những gì bạn đã làm và có thể note những thứ nhỏ mà bạn thích một cách đơn giản. Căn bản là mình làm cái này để làm report daily cho sếp rất hữu dụng mà còn đơn giản nữa.
