Create Blog Association Between GitHub Page and Hexo - Blog Framework

2015-05-21

Đây là bài blog đầu tiên của mình về cách tạo ra một blog đơn giản static page cho dân IT (giống như cái này http://khanhkid.github.io/blog/), Các bạn có thể tự tạo cho mình 1 blog căn bản, quản lý bài viết, đăng bài bằng Mark Down 2, cũng như gia nhập vào thế giới bloger. :)

Chú ý: Vì cái này là dạng dùng static page (không dùng PHP và Database) nên không có vụ có editor gì đó bấm phát đăng bài đâu nhé. Toàn bộ là code cả :D.

Tool Requirement

  • Account GitHub: Vào đây tạo 1 Account GitHub https://github.com/
  • Git Bash : Cái này là công cụ để up code lên trang GitHub
  • Node.js: Cái này dùng để cài đặt Hexo
  • MarkDown 2: Cái này là dùng để viết bài. Cái này hiện nay dùng khá nhiều với đuôi là .md -> xưa mấy cái Readme thường là .txt giờ người ta dùng cái này để viết Readme. Viết xong nó có thể cover qua HTML cho mình luôn.

Page Reference:

  • GitHub Page: Cách tạo page trên 1 cái Repository trên GitHub
  • Hexo: Dùng để tạo 1 thư viện để Generate từ cái file Mark Down (*.md) thành 1 trang blog như vầy http://khanhkid.github.io/blog/

Overview Step by Step:

Mình overview căn bản cho các bạn tìm hiểu cho dễ. Đối với những bạn không làm được thì ráng đợi mình sẽ viết các bài tiếp theo để hướng dẫn từng bước step by step cho dễ hình dung. :)

Cơ bản nó sẽ gồm những Step chính như vầy:

Step 1: Tạo cái Account của GitHub.

Step 2: Develop code cái “Hello World” lên GitHub. Cho quen môi trường GitHub

Step 3: Tạo cái Page cho cái GitHub của mình. Làm theo cái này nè GitHub Page.

Step 4: Cài cái Node.js và cài Hexo

Step 5: Cách generate ra 1 trang blog từ file Mark Down.
